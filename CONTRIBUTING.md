# Contributing to Vibe Coding Workshop

Welcome! To participate in the session and earn your badges, please follow these steps:

1. **Fork & Clone:** Fork this repository and clone it to your local machine.
2. **Branching:** Create a branch named `participant/[your-name]-[city]`.
3. **Commit Messages:** Follow the standard: `[UC-ID] Fix [what]: [why it failed] → [what you changed]`.
4. **Pull Request:** Open a PR against `main` with the title `[City] [Name] — Vibe Coding Submission`.

Ensure all 4 User Cases (UCs) are completed to be eligible for the completion badge!
